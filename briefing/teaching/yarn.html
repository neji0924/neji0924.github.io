yarn

需熟悉npm

什麼是yarn

類似npm

為了改善npm而產生的

參考連結:比NPM更快！Facebook開源JavaScript套件管理工具YARN

yarn懶人包

  • 解決npm速度慢而產生
  • 並非取代npm
  • 依然拉取https://www.npmjs.com上的package
  • 預設產生.lock (npm5.0後預設也是會產生了)
  • 並行安裝
  • 離線安裝

CLI差異

npm yarn
全域 -g / --global global
install package.json yarn.lock
update npm update yarn upgrade
add [-dev] --save-dev -dev
END.
