npm教學

npm是什麼

npm為node.js的套件管理工具

node.js在0.6.3後內建npm

安裝

直接安裝node.js

windows

Download | Node.js

MAC

$ brew install node

或是安裝nvm再由nvm安裝node

測試一下是否可以執行

$ npm -v

指令教學

先看過就好不用操作

全域安裝

$ npm install -g {package name}

專案安裝

$ npm install {package name}

記得先cd到專案目錄下

移除套件


$ npm uninstall -g {package name} //全域
$ npm uninstall {package name} //專案套件

列出套件


$ npm list -g  //全域
$ npm list     //專案套件

還有很多指令

要用到的時候在查即可

package.json

好用的來了喔

建立

cd至專案跟目錄

$ npm init

Enter大法後就會產出package.json

有點類似安裝的時候下一步下一步下一步

package.json樣貌

(用sublime開啟即可)


{
  "name": "test",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "dependencies": {},
  "devDependencies": {},
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "ISC"
}

簡介

dependencies: 上線會用到的套件

devDependencies: 開發會用到的套件

其他都不重要

如何安裝package.json裡面的套件

$ npm install

如何只安裝開發套件

$ npm install --only=production

範例中只有devDependencies下哪個指令都可以

套件資料夾

node_modules

有使用git記得忽略掉這個資料夾

或是要包成zip給人的時候也是

安裝套件方式

下指令安裝後把安裝的套件記錄在json裡

$ npm install {package name} --save
$ npm install {package name} --save-dev

-dev就不多說了

--save沒加不會將資訊存在package.json裡面
