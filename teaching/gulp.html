gulp教學

gulp是什麼

gulp是運行在node.js上的一套js

專門自動化處理前端的雜事

主要做哪些雜事?

  • 壓縮
  • 合併

包含圖片、html、js、css、文件

還有許多功能可自行上網搜尋

安裝

全域安裝gulp

$ npm install -g gulp

測試一下是否可以執行

$ gulp -v

gulpfile.js

要執行gulp必須在專案根目錄下建立這隻檔案

裡面寫你想要執行的task

此篇教學略過

Laravel-elixir

不會寫task怎麼辦

Laravel-elixir來拯救你

要開始實做了喔

建立專案

首先建立一個練習gulp的資料夾

並建立npm 的package.json (提示:init)

package.json


{
  "devDependencies": {
    "bootstrap-sass": "^3.3.7",
    "gulp": "^3.9.1",
    "jquery": "^3.1.0",
    "jquery-ui-dist": "^1.12.1",
    "laravel-elixir": "^6.0.0-14",
    "laravel-elixir-vue-2": "^0.2.0",
    "laravel-elixir-webpack-official": "^1.0.2",
    "font-awesome": "*",
    "compass-mixins": "^0.12.10"
  }
}

先複製貼上這個package.json (本次教學用到的套件)

安裝

$ npm install

會多node_modules資料夾

gulpfile.js

此篇說明gulpfile.js

先建立gulpfile.js


var elixir = require('laravel-elixir');

elixir((mix) => {
    mix.copy('./node_modules/font-awesome/fonts', 'public/fonts');
  
    mix.sass('app.scss');

    mix.scripts([
        './node_modules/jquery/dist/jquery.min.js',
        './node_modules/jquery-ui-dist/jquery-ui.min.js',
        './node_modules/bootstrap-sass/assets/javascripts/bootstrap.js',
        'app.js'
    ], 'public/js/app.js');
});

複製貼上此範例

簡介

建立資料夾

resources/assets/sass/app.scss

resources/assets/js/app.js

此為laravel-elixir架構

不想要laravel的架構?

在gulpfile.js說明檔案路徑

mix.sass('./app.scss', './app.css');

直接將專案根目錄的scss編譯成css

可自行玩看看不過還是建議遵循結構

多檔案

有兩隻scss怎麼辦呢

編譯成兩隻css


mix.sass('app.scss');
mix.sass('app2.scss');

編譯成一隻css


mix.sass(['app.scss', 'app2.scss']);

那每次增加檔案都要再去修改gulpfile?

匹配符號來囉


mix.sass('*.scss')  // 所有scss
mix.sass('./resources/assets/sass/**/*.scss') //若你的scss有依照專案分資料夾
mix.sass('!aaaa.scss')	// 除了 aaaa.scss以外
mix.sass('{aaa,bbb}.scss') // 有在這裏面的

大概知道一下就好有需要的時候在來翻

gulp執行

終於要開始了

目前專案目錄

開發時

寫好需要執行一次任務

$ gulp

每次存檔都會執行任務

$ gulp watch

目前專案目錄

多了一個public資料夾

你的成果都在裡面了

html裡面也是匯入public的東西

專案上線前

$ gulp --production

你的public裡面的東西都被壓縮了

享受gulp帶給你的便利吧!!
